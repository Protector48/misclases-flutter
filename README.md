🚀 Descripción del Proyecto

Este repositorio contiene ejercicios prácticos desarrollados en Flutter y Dart, orientados al aprendizaje de interfaces móviles, control de listas, scroll dinámico, consumo de APIs REST,  manejo de estado, etc.

El proyecto tiene como finalidad reforzar conocimientos prácticos sobre el desarrollo de aplicaciones móviles multiplataforma usando Flutter.

⸻

✨ Funcionalidades Implementadas

✅ ListView
✅ ListView Builder
✅ ScrollController
✅ Scroll Infinito
✅ Listas Dinámicas
✅ Consumo de APIs REST
✅ Integración con PokéAPI
✅ Widgets en Flutter
✅ Manejo de Provider
✅ Navegación entre pantallas.  y mass
